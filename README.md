Penjelasan sistem singkat: 
Stationify adalah sebuah toko yang menyediakan alat tulis, dimana sistem ini dibuat sebagai sistem pengelolaan barang yang ada di stationify agar lebih efisien dan juga terstruktur. Fungsi utama dari sistem ini adalah untuk mensimulasikan sistem manajemen barang alat tulis (ATK) tanpa menggunakan database, sehingga semua data masih berupa array yang diolah di dalam controller.

Fitur: 
1. Login
2. Dashboard
3. Profile
4. Pengelolaan Barang
5. About Stationify
